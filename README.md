##### 工程目录文件
> - **yarn.lock**   
项目依赖安装包的版本号（不用动）
> - **package.json**          
代表该脚手架是一个node的包文件，包含了项目的介绍
> - **.gitignore**     
用git进行版本控制的时候不进行上传的文件
> - **node_modules**   
项目第三方依赖的包
> - **public**    
favicon.ico:  显示在项目标题前面的图标   
index.html:  项目首页的html模板
> - **src**  
所有的项目源代码    
**index.js：** 项目入口（整个程序运行的入口文件）
